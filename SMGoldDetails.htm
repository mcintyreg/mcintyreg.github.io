Soccer Manager





The game begins from a central hub/terminal which allows players to choose from options for line-up selection, transfers, selling of players, tactic selection and also saving and loading of a game.

This project looks dated now and doesn't have the nicest interface. For instance line-up selection is done by entering names from list. Quite why I did it this way (as opposed to entering shirt numbers) I don't remember. It was however the first experience I got of loading and saving from a database type file and calculating random results based on a number of factors. These include match results, goals and scorers and transfer prices and players.

In addition to this assembly routines were used which allowed me to receive input from the mouse, again as this is not something which you normally get using the QBasic compiler.

The game also had to fit into a small amount of memory in comparison to game these days, and since this kind of game really needs a large database this was a challenge.

The game includes an install program which was used in the past to install from a floppy disc.
