1.  Installation

To install Omega Strike just unzip the zip file into a directory on your Hard Drive.

2. Requirements

Windows operating system
386 or better processor
2MB of RAM
VGA 256 colour graphics

Omega Strike has to be run from DOS or a dos box in Win95/98
Newer operating systems such as Windows XP/Vista/Windows 7 will require and emulator such as DOSBox

3. Launching

It is likely that this game will need to be run in an emulator under later operating systems due to the fact it is so old now. The DOS box emulator is a good one and reasonably easy to use. Once installed you need to use the following commands to run the game.

     mount j C:\Os_Alpha
     j:
     omega

Replace the path with the install path or folder name where the game has been downloaded to (copying to C: is probably the easiest thing to do)

The default CPU Speed/Cycles for the DOS emulator is a little low so use Ctrl-F12 to set this up to 8000 or more for better performance.


To play Omega Strike run Omega.exe
To run the Level Designer run Osedit.exe

When at the menu you will have the choice of either to load or start a new campaign or to play a custom game. There are two types of custom game you can play, you can either play a premade Scenario, play one you have made yourself, or you can play a skirmish game.

4. Controls

Omega Strike is a turn based game. In a turn you can move and attack with all of your units and you can build more units or structures. Units have maximum range (distance they can move) and a limted number of attacks which they can make in one turn. What you can produce in one turn is determined by your Action Points (AP's) and Credits. Training soldiers does not cost you AP's, but everything else does. Building vehicles takes 1 AP and building any structure takes 2 AP's.

Basic controls

Unit/structure production requires key commands, these are as follows;

B - Build unit/structure +
  O - Ore truck
  M - Mech infantry
  L - Light tank (N.A.T.O only)
  T - T110 Battle tank (N.A.T.O only)
  S - Surface to surface missile launcher (Rebels only)
  D - Battle droid (Rebels only)
 
  R - Refinery
  B - Barrack
  P - Power plant
  C - Communication centre
  G - Gun turret
  L - Lazer tower (Rebels only)

Buildings can be placed using the mouse

T - Train unit +
  S - Soldier
  C - Commando (N.A.T.O only)
  E - Engineer

Other useful key commands;

H - Bring this help menu up in game

O - Set ore truck to collect ore
X - Fire orbital lazer cannon/launch nuke (when charged)
` - Save Game

Unit actioning is mainly mouse controlled, but actions also require a key command, these are as follows;

Once unit has been selected with mouse;
M - Move mode (use cursor keys and enter to stop)
A - Attack mode (select a target using mouse)
Note: Enter completes a units move!

Advanced controls

Escape will deselect a unit.

Pressing " / " will take you to the console. From the console the following commands maybe useful;
    save - Save Game
    load - Load Game
    brief - Mission briefing (if playing a campaign)
    surrender - Restart mission
    quit - Exits game
   (Some of the cheats in the game are typed from here too)

Repair of buildings and units (unit needs to be close to factory) can be done using the button with a spanner icon at the bottom of the screen.

Note: Scrolling the map is done via scroll bars at the bottom and right of the screen. The scroll speed is set from Options on the main menu, this will change the scroll speed in the game and also in the level editor.

5. Buildings

N.A.T.O. Structures

Factory (Cannot build)
VERY IMPORTANT! This is where materials are kept and everything from structures to tanks are built. If you loose this building then you will not be able to build anything.

Refinery
This is the place which refines the ore collected by ore trucks and turns it into the many different minerals it's made up of so that they can be used for building. If you loose this building your income will be cut. An ore truck does not come with the refinery.
(700 credits)

Barracks
This is the place where soldiers are trained. If you loose this structure then you will not be able to train more men.
(300 credits)

Power Plant
The centre whic supplys electricity to your buildings, if this place is lost then production in other facilities will be stopped.
(250 credits)

Communication Centre
This structure is the control centre for the Orbital Lazer Cannon. Once this is built the Lazer will start to charge. It takes a few turns but once ready you can fire it at anything no matter where it is.
(1250 credits)

Gun Turret
This structure is for defence. Use it to fire explosive shells at enemies units which stray into it's path. (400 credits)

Rebel Structures


Factory (Cannot build)
VERY IMPORTANT! This is where materials are kept and everything from structures to tanks are built. If you loose this building then you will not be able to build anything.

Refinery
This is the place which refines the ore collected by ore trucks and turns it into the many different minerals it's made up of so that they can be used for building. If you loose this building your income will be cut. An ore truck does not come with the refinery so be careful when spending credits.
(700 credits)

Barracks
This is the place where soldiers are trained. If it is destroyed then you will not be able to train more men. (300 credits)

Power plant
This centre supplys electricity to your buildings, if it is destroyed then production in other places will be halted.
(250 credits)

Communication Centre
This structure is the control centre for the launch of the nuclear missile. Once this is built construction of a nuclear weapon starts. Once the missile is built then you will be able to launch it. This is the most powerful unit in the game.
(1250 credits)

Gun Turret
This structure is for defence. Use it to fire explosive shells at enemies units which stray into it's path. (400 credits)

Laser Tower
This structure is for defence. It will fire a lazer beam at any enemies units which stray into it's path. It is slow but incredibly powerful. It also requires power.
(600 credits)


6. Units

N.A.T.O Forces


Soldier
Man, lightly armoured carrying machine gun. Men are formed into squads of up to 5, squads always stay together. Training a new soldier requires the squad to return to base.
(50 credits each)

Commando
This guy carries a sniper rifle which can take out men with ease from range. However, his rifle doesn't have the capability to last long against vehicles or buildings.
(200 credits)

Engineer
Use this guy for capturing enemy buildings. He doesn't have a weapon so guard him carefully. (Also note he cannot collect crates ).
(250 credits)

Mech Infantry
This jeep is the fastest unit in the game, also being faster than the Rebel's version. Although for this speed boost it sacrafices a little armour. It's armed with a high powered machine gun.
(175 credits)

Light Tank
This tank has moderate armour and a moderate movement rate. It fires small explosive shells into enemy ranks piercing through lightly armoured units like cheese.
(300 credits)

T-110 Battle Tank
This heavly armoured tank fires high powered shells into enemy ranks destroying virtually anything in it's way.
(500 credits)

Ore Truck
This vehicle collects ore and takes it to your refinery. It can be attacked so guard it. To set it collecting ore press "O" and click on an ore mine.
(400 credits)

Orbital Laser Cannon

This weapon charges faster than the nuke but does not have the power of it's big brother.
(Communication Centre required)


Rebel Forces

Soldier
Man, lightly armoured carrying machine gun. Men are formed into squads of up to 5, squads always stay together. Training a new soldier requires the squad to return to base.
(50 credits each)

Engineer
Use this guy for capturing enemy buildings. He doesn't have a weapon so guard him carefully. (Also note he cannot collect crates )
(250 credits)

Mech Infantry
Although this mechanised unit is slower than it's N.A.T.O. counter part it is better protected. It's armed with a high powered machine gun.
(175 credits)

SSM Launcher
This unit comes equipped with powerful ICBM missiles, so powerful that even a tanks armour will suffer heavy damages. The draw backs are it's reload time, taking a full turn to reload and it's armour which is poor. They have quite a good range.
(350 credits)

Omega Battle Droid
This walking tower is armed with a laser weapon on each arm which will fry anything in its path. To add to this it has fairly strong armour too. You may not command more than one of these units at once.
(750 credits)

Ore Truck
This vehicle collects ore and takes it to your refinery. It can be attacked so guard it. To set it collecting ore press "O" and click on an ore mine.
(400 credits)

Nuclear Missile

This super weapon takes ages to charge, but when it has NOTHING will be left of the enemy!
(Communication Centre required)
