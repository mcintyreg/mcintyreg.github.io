Texas Hold 'Em Poker


Game against two computer opponents


Settings dialogue


Chat feature

I developed a texas hold-em based poker game. You can customise difficulty of the AI and how much money each players starts with along with tweaking the game rules. Up to 3 AI opponents are available, which are picked randomly from a handful of default players with different profiles.

Betting options appear in the bottom left and blinds are allocated based on where the dealer sits, a red button indicates which player is currently to play. The game is not too difficult to pick up if you know how to play the game although the rules page in the help may not be enough for a new player to understand how to play.

UML was used to design the game classes for this project as part of the work (see above)

This project improved my knowledge of working with MFC, as my experience before this point had been base in Win32 natively or through using the .NET framework.



Note: this game has been tested under both XP and Windows 7 (64bit) but it hasn't been tested in earlier operating systems.
