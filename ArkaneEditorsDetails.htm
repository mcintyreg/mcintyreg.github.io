Arkane Editors

I developed a suite containing two editors for my RPG game project. The first editor is a graphical tool written using the engine and OpenGL to aid in map creation. Users can add river sections, hills, buildings, trees and bushes and even smaller items for dungeon map creation. There is a mini map for quick navigation and a full screen preview. The tool also has a snapping grid to aid with placement, and a house interior designer, which allows buildings to be populated with furniture. You can also create quest packs and link them to things on the map.

Behind the scenes the tool outputs a map file, XML and batch file for the texture packages to be built for the game to use. My texture atlasing tool packs these into texture DDS files which are stored inside a package file.





The second editor allows you to add items, spells, creatures, item making recipes, potion recipes and dialogue; these are contained in a pak file which can then be loaded by the game. The tools are intended to be used to catch problems with the engine early in development, in addition to allowing designers to create the content. The data tool is used for graphic file linking as well as statistics data entry for balancing. It has helped me think about the game design in advance of implementation.
