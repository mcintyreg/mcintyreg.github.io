Arkane


Download design document here
Language/IDE: C++ OpenGL / Visual Studio 7/8
Project Duration: Feb 2010-Ongoing

I am currently undertaking a larger project to create an RPG along with a toolset.

I intend to implement random dungeon generation, path finding using A*, a potion creation system based on collecting ingredients, a weapon making system along with a simple to use interface. Click on the image or title to see more screen shots and details. The games unique element will be the story.

Much of the required artwork for the world, items and characters is complete, I am working with an artist to create UI elements. I have completed some tools (see Personal Projects) to create items, spells and creatures, and a map editor to build the world. There is also a designer and script writer working on the project


















3D Castle Scene

See main page here

Language/IDE: C++ OpenGL / Visual Studio 7/8
Project Duration: Jan 2009-October 2010

In future I intend to add geometry shaders to my castle project, the idea being that these will be used to reduce the number of instances I need when grass is rendered in more detail (i.e. from a series of models as opposed to a single billboarded texture)

I am also looking to improve the smoke effects using vortex particles to create volumetric effects and adding a weapon and using the parallax to render bullet holes in the walls

I will also be looking to add shadow mapping and there are some minor issues which need resolving such as lens flare occlusion.
